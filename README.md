## Skills

### java-interview-agent

专业的 Java 后端开发面试模拟器。基于候选人简历（PDF）进行针对性技术面试，覆盖：

- Java 基础 / JVM / Spring 框架
- 中间件（RocketMQ、Kafka）/ 数据库（MySQL、Redis、ES）
- 项目经验深入追问
- 系统设计
- AI / Agent / MCP 相关

**特性：**
- 一问一答模式，由浅入深
- 70% 问题与简历项目相关
- 面试结束自动生成总结报告（含完整参考答案）
- 历史记录追踪，避免重复提问，定向抽查薄弱项
